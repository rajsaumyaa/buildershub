## 🚀 What is BuildersHub India?

BuildersHub India is an AI-native community platform built specifically for Indian developers. It combines competitive leaderboards, intelligent AI-powered mentorship, and community-wide AI adoption tracking — all orchestrated through an agentic architecture using **Superplane**, **Zynd AI**, and **Apify**.

Unlike generic hackathon platforms, BuildersHub is self-running. Three specialized AI agents handle mentorship, leaderboard management, and AI impact tracking autonomously — with Superplane wiring all pipelines together end-to-end.

## ✨ Features

### 🏆 Leaderboard
- Developers register with their GitHub handle
- **Apify** scrapes real public GitHub stats (repos, followers, contributions) via the GitHub Scraper Actor
- Score is auto-calculated and seeded to the leaderboard instantly
- Rankings update in real time after every event and mentorship session

### 🤖 AI Mentorship (Zynd MentorAgent)
- Developers paste code and describe their problem
- **Zynd MentorAgent** — registered on the Zynd AI agent network — processes the request
- Review is generated by **LLaMA 3.3 70B via Groq** with 5 structured sections: Summary, Strengths, Issues, Suggestions, Score
- Session is logged to the database and 20 bonus points are awarded to the developer

### 📊 AI Impact Dashboard
- Tracks community-wide AI tool adoption across Indian builders
- Displays GitHub Copilot usage, AI-assisted commits, and MentorAgent sessions
- Weekly trend chart powered by Recharts
- Encourages healthy competition around AI adoption, not just raw coding speed

## 🔗 Sponsor Tool Integration

### Superplane — Core/Essential
Superplane orchestrates all event-driven workflows across the platform:
- **Registration Pipeline:** User registers → Apify fetches GitHub stats → Score seeded → Leaderboard updated
- **Mentorship Pipeline:** Code submitted → MentorAgent reviews → Points awarded → Session logged

### Zynd AI — Core/Essential
MentorAgent is registered on the Zynd AI agent network:
- Accepts code review requests from the platform
- Calls LLaMA 3.3 70B via Groq to generate structured feedback
- Returns 5-section review: Summary, Strengths, Issues, Suggestions, Score
- Awards 20 points per completed session

### Apify — Core/Essential
Apify powers real data enrichment across the platform:
- GitHub Scraper Actor fetches public developer stats on registration
- Calculates initial leaderboard score from repos and followers
- Enables data-driven rankings based on real builder activity

### GitHub Copilot — Moderate
- Used throughout development to accelerate feature building
- Tracked as a community metric on the AI Impact dashboard

## 🏃 Running Locally

### Prerequisites
- Python 3.10+
- Node.js 18+
- Docker (for Superplane)
- PostgreSQL (Railway free tier)

### Backend
```bash
cd buildershub-api
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend
```bash
cd buildershub
npm install
npm run dev -- --port 3001
```

### Superplane
```bash
docker run --rm -p 3000:3000 -v spdata:/app/data -ti ghcr.io/superplanehq/superplane-demo:stable
```

### Environment Variables
Create `.env` in `buildershub-api/`:
```env
DATABASE_URL=your_postgresql_url
APIFY_TOKEN=your_apify_token
GROQ_API_KEY=your_groq_key
ZYND_API_KEY=your_zynd_key
```
